Muzzler.Windows
=======

Muzzler provides windows graphical interface by using Aurora, a hardware accelerated graphical interface library. The Aurora documentation can be found [here](http://muzzler.github.io/Muzzler.Windows/).

Conventions
=======

1. All C/C++/H files will be capitalized exccept `stdafx` and `targver`.
2. Library-side folders will be capitalized.
3. Client-side folders will be lowercased.
4. `camelCase` for functions and variables.
5. `A2DCamelCase` for classes.
